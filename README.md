# 🎮 Tic Tac Toe Game (C++)

## 📌 Description
This project is a console-based Tic Tac Toe game developed using C++. It demonstrates core programming concepts such as loops, arrays, and conditional logic.

## 🚀 Features
- Two-player game (X and O)
- Dynamic board display after each move
- Win detection
- Draw detection
- Replay option

## 🛠 Technologies Used
- C++

## ▶️ How to Run
1. Open the code in Cxxdroid or any C++ compiler
2. Run the program
3. Enter positions from 1 to 9 to play
